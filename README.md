# Comic Clone

Free comics website using Next + Tailwind </br>
Clone from website use Nuxt and API from https://comics-api.vercel.app/

## Official Link

https://comic-clone.vercel.app/

## Technology

- NextJS
- TailwindCSS
- Swiper
- Other libraries

## Features

- Fully responsive
- Save/delete read history
